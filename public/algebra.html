Algebra

Learn Algebra in a simple and practical way. This page contains theory, detailed explanations, formulas, and examples for each topic.

1. Basic Algebra

Algebra is a branch of mathematics that uses symbols (like x, y, z) to represent numbers. It allows us to generalize arithmetic operations and solve problems systematically.

Theory:

In basic algebra, expressions, equations, and variables are used to describe patterns and relationships.

Example:

x + 5 = 10 → x = 10 - 5 → x = 5

Here, x represents the unknown number.

2. Linear Equations

Linear equations are algebraic equations in which the variable has a power of 1. They form a straight line when graphed.

Theory:

A linear equation in one variable has the form: ax + b = 0, where a ≠ 0.

Example:

2x + 3 = 7 → 2x = 4 → x = 2

This shows how to isolate x to solve the equation.

3. Quadratic Equations

Quadratic equations are equations where the highest power of the variable is 2. They are important in physics, engineering, and real-world problem-solving.

Theory:

A quadratic equation has the general form: ax² + bx + c = 0, where a ≠ 0.

Solution:

x = [-b ± √(b² - 4ac)] / 2a

Example:

x² - 5x + 6 = 0 → x = 2 or x = 3

4. Factorization

Factorization is expressing an algebraic expression as a product of simpler expressions (factors). It simplifies solving equations.

Theory:

Factorization techniques include:

  • Common factor extraction
  • Difference of squares
  • Trinomial factorization

Example:

x² + 5x + 6 = (x + 2)(x + 3)

5. Algebraic Identities

Algebraic identities are formulas that are always true for all values of variables. They simplify calculations.

Important Identities:

  • (a + b)² = a² + 2ab + b²
  • (a - b)² = a² - 2ab + b²
  • a² - b² = (a - b)(a + b)
  • (x + a)(x + b) = x² + (a + b)x + ab

Example:

(x + 3)² = x² + 6x + 9

6. Systems of Equations

Systems of equations consist of multiple equations with multiple variables. The goal is to find values of variables that satisfy all equations simultaneously.

Example:

x + y = 10
2x - y = 3
Solve: x = 13/3, y = 17/3

7. Polynomials

Polynomials are expressions with one or more terms, consisting of variables and coefficients combined using addition, subtraction, and multiplication.

Example:

P(x) = 2x³ + 3x² - x + 5

8. Exponents & Powers

Exponents represent repeated multiplication of the same number. xⁿ means multiplying x by itself n times.

Example:

2³ = 2 × 2 × 2 = 8

9. Inequalities

Inequalities show the relationship between expressions that are not equal. Symbols: >, <, ≥, ≤

Example:

x + 3 > 7 → x > 4
