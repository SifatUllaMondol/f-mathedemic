Permutation & Combination

Complete theory, examples, and detailed explanations from basic to advanced level.

1. Permutation Theory

Permutation refers to the arrangement of objects in a specific order. Order matters in permutation. 

Number of permutations of n distinct objects = n!
Permutation of r objects from n = P(n, r) = n! / (n-r)!

Example: Arrange 3 letters A, B, C → 3! = 6 ways: ABC, ACB, BAC, BCA, CAB, CBA

2. Combination Theory

Combination refers to the selection of objects where order does not matter. 

Number of combinations of n objects taken r at a time = C(n, r) = n! / (r!(n-r)!)

Example: Select 2 letters from A, B, C → C(3,2) = 3 ways: AB, AC, BC

3. Advanced Concepts

3.1 Permutation with Identical Objects

When objects are identical, total arrangements decrease: 

Permutation = n! / (p1! p2! ... pk!)

Example: Word "SUCCESS" → S=3, C=2, U=1, E=1 → 7! / (3!2!1!1!) = 420

3.2 Permutations with Restrictions

Case 1: Certain elements together → treat as a single block. Example: Arrange A, B, C, D with A & B together → Total = 12 ways Case 2: Certain elements not together → Total − arrangements with together.

3.3 Circular Permutations

Circular permutations formula: (n-1)! If identical objects, divide by factorials of identical objects. Example: A, A, B around circle → (3-1)! / 2! = 1 way

4. Examples & Explanations

1. How many 3-letter codes can be formed from A, B, C, D, E if no letter repeats? Solution: P(5,3) = 5 × 4 × 3 = 60

2. How many ways can a team of 2 boys and 2 girls be formed from 4 boys and 3 girls? Solution: C(4,2) × C(3,2) = 6 × 3 = 18

3. Arranging letters of "MISSISSIPPI": M=1, I=4, S=4, P=2 → 11! / (1!4!4!2!) = 34650
